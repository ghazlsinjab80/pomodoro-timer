 🍅 Pomodoro Focus Timer

A feature-rich Pomodoro timer built with vanilla JavaScript to boost productivity and focus.

 Features

- Customizable Sessions: Set your own work and break durations
- Dark/Light Mode: Easy on the eyes, day or night
- Audio Notifications: Get alerted when sessions change
- Session Tracking: Counts completed pomodoros automatically
- Persistent Settings: Your preferences are saved in your browser
- Fully Responsive: Works on desktop, tablet, and mobile

 Tech Stack

- HTML5 - Semantic structure
- CSS3 - Flexbox, Grid, animations, custom properties
- Vanilla JavaScript - ES6+ classes, DOM manipulation
- Web Audio API - Notification sounds
- localStorage - User preferences persistence
- Browser Notifications API - Desktop alerts

 Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/pomodoro-timer.git
```

2. Open `index.html` in your browser

3. Start focusing

How to Use

1. Set your preferred work and break durations using the input fields
2. Click "Start" to begin your focus session
3. The timer will automatically switch between work and break periods
4. Track your completed sessions in the statistics section
5. Toggle dark mode for comfortable evening work
